# 구글 머티리얼 디자인(Material Design 2) 다크 테마 가이드

이 가이드는 구글 머티리얼 디자인 가이드를 기반으로, 저조도 환경에서 사용자 경험을 최적화하고 배터리 효율을 높이기 위한 다크 테마 설계 원칙을 정리합니다.

---

## 1. 다크 테마의 핵심 원칙 (Principles)

* **Dark Grey 사용:** 순수 검정(#000000)보다는 짙은 회색을 기본 배경으로 사용합니다. 이는 그림자(Shadow)를 통해 깊이감을 표현하기에 더 유리하며, 눈의 피로를 줄여줍니다.
* **컬러 강조 (Color with Accents):** UI 전체에 색상을 채우기보다는 채도가 낮은(Desaturated) 색상을 포인트로 사용하여 가독성을 높입니다.
* **에너지 절약 (Conserve Energy):** OLED 디스플레이 기기에서 어두운 색상은 전력 소모를 줄여 배터리 수명을 연장합니다.
* **접근성 보장 (Accessibility):** 모든 텍스트와 배경 간의 대비는 최소 **4.5:1** 이상의 대비 비율을 유지해야 합니다.

---

## 2. 고도와 표면 (Elevation & Surfaces)

다크 테마에서는 그림자가 잘 보이지 않기 때문에, **표면의 밝기(Brightness)**를 통해 계층 구조를 표현합니다.

### 고도에 따른 표면 색상 (Surface Overlays)
고도(Elevation)가 높아질수록(사용자에게 가까워질수록) 표면에 투명한 흰색 오버레이를 겹쳐 더 밝은 회색으로 표현합니다.

| Elevation (dp) | Overlay Opacity | 추천 Hex Color (Base: #121212) |
| :--- | :--- | :--- |
| **0dp** | 0% | `#121212` |
| **1dp** | 5% | `#1E1E1E` |
| **4dp** | 9% | `#272727` |
| **8dp** | 12% | `#2E2E2E` |
| **16dp** | 15% | `#353535` |
| **24dp** | 16% | `#383838` |

> **Tip:** 다크 테마에서도 그림자(Shadow)는 여전히 사용됩니다. 하지만 표면의 밝기 변화가 계층을 구분하는 주된 역할을 합니다.

---

## 3. 컬러 팔레트 (Color Palette)

라이트 테마의 선명한 색상을 다크 테마에서 그대로 사용하면 눈에 자극을 줄 수 있습니다.

### 채도 조절 (Desaturation)
* 다크 테마에서는 채도가 낮은 **200~50 범위**의 색상을 권장합니다.
* 채도가 높은 색상은 어두운 배경에서 시각적 진동(Optical Vibration)을 일으켜 눈을 쉽게 피로하게 만듭니다.

### 주요 색상 적용
* **Primary Color:** 라이트 테마보다 밝고 채도가 낮은 톤을 사용합니다.
* **Error Color:** 기본 에러 색상(#B00020) 대신, 다크 테마용으로 밝기가 조정된 **#CF6679**를 주로 사용합니다.
* **On-Colors:** 배경 위에 올라가는 텍스트나 아이콘은 주로 흰색(#FFFFFF)을 사용하되, 중요도에 따라 투명도를 조절합니다.

---

## 4. 텍스트 및 가독성 (Typography)

텍스트는 배경과의 대비가 명확해야 하며, 흰색 텍스트의 불투명도를 조절하여 계층을 나눕니다.

| 강조 수준 (Emphasis) | 불투명도 (Opacity) | 용도 |
| :--- | :--- | :--- |
| **High Emphasis** | 87% | 제목, 본문 등 주요 정보 |
| **Medium Emphasis** | 60% | 보조 설명, 라벨 |
| **Disabled Text** | 38% | 비활성화된 상태 |

---

## 5. 디자인 체크리스트 (Best Practices)

- [ ] **순수 검정 지양:** 배경색으로 `#121212`를 사용했는가?
- [ ] **색상 반전 금지:** 단순히 라이트 테마의 색상을 반전(Invert)시킨 것은 아닌가?
- [ ] **고도 표현:** 고도가 높아질수록 표면이 적절하게 밝아지는가?
- [ ] **대비 확인:** 텍스트와 배경의 대비가 4.5:1(WCAG AA)을 만족하는가?
- [ ] **브랜드 아이덴티티:** 브랜드 컬러가 다크 테마에서도 명확하게 식별되는가?
